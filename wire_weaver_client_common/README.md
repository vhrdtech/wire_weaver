# wire_weaver_client_common

![Crates.io Version](https://img.shields.io/crates/v/wire_weaver_client_common)

> Data types and common code for WireWeaver API clients on host side (WebSocket, UDP, USB)

There is no need to use this crate directly.
See [wire_weaver_template](https://github.com/vhrdtech/wire_weaver_template) showcasing how to use WireWeaver on
real hardware, connected via USB.
